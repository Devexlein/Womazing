# Womazing

Выпускной проект: Адаптивная верстка сайта онлайн-магазина одежды:
- верстка + интеграция с Redux + интеграция страниц “Магазин”, “Один товар”, “Корзина”, “Оформление заказа” и “Успешный заказ” с Woocommerce,
- с использованием Bootstrap Grid,
- анимация всех интерактивных элементов (меню - на разрешениях ниже 992px преобразуется в мобильное меню-гамбургер, стрелки, кнопки, ссылки), два слайдера на главной странице,
- оверлей для всех карточек товара с переходом на страницы товара/магазина,
- реализованы 3 формы для ввода данных с валидацией корректности их заполнения (на странице Контакты валидация формы обратной связи через плагин Contact Form 7), модальное окно с формой заказа обратного звонка. 

Сайт интегрирован в WordPress (Redux Framework: редактирование содержимого страниц через админпанель; WooCommerce, а именно: просмотр страницы одного товара, добавление в корзину, оплата).

Заказчик: Проект в рамках обучения WayUp, курс "Веб-верстальщик"
