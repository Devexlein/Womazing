# Womazing


Задание: Адаптивная верстка сайта онлайн-магазина одежды с помощью Bootstrap Grid и анимация всех интерактивных элементов 
(меню - на разрешениях ниже 992px преобразуется в мобильное меню-гамбургер, стрелки, кнопки, ссылки), два слайдера на главной странице,
оверлей для всех карточек товара с переходом на страницы товара/магазина,
реализованы 3 формы для ввода данных с валидацией корректности их заполнения (на странице Контакты форма обратной связи реализована с помощью плагина Contact Form 7), модальное окно с формой заказа обратного звонка. 
Сайт интегрирован в WordPress (Redux Framework: редактирование содержимого страниц через админпанель; WooCommerce, а именно: просмотр страницы одного товара, добавление в корзину, оплата)

Заказчик: Проект в рамках обучения WayUp

Макет: https://www.figma.com/file/iOyjVvSOadRgau9sQE3iAZ/3Dcommerce
